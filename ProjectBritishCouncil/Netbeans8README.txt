To run this project on Netbeans 8
------------------------------------
1) Open the project in Netbeans 8
2) Right click the Project Name on the side bar, click Properties
3) The properties for the project should appear. Now select Libraries
4) You'll see an error showing something as missing. Just select the combo box and change to an existing Java library (probably JDK8/Default something)
5) Click on Apply/Save and the project should work now
